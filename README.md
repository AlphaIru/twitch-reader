## Update

2026 April 08

We are splitting into three branches:

- main
- yomi
- nico

### main

This is the main branch; this will include all features that other branches combined.

### yomi

This branch will only include TTS by OpenJTalk. If the streamer only wants to hear the chat, this is the branch to use.

### nico

This will utilize JavaScript to display the chat like Niconico. If the streamer wants to display the chat to the viewers, this is the branch to use.
